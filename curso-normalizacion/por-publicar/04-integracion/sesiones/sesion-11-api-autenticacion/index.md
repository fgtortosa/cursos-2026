# Sesión 11: Llamadas a la API y autenticación

[[toc]]

::: info CONTEXTO
En las sesiones anteriores aprendimos a crear APIs (.NET) y componentes (Vue) por separado. Ahora conectamos ambas partes: cómo Vue llama a la API, cómo sabe el servidor quién es el usuario y cómo exploramos la API de forma interactiva.
:::

## Objetivos

Al finalizar esta sesión, el alumno será capaz de:
- Entender el flujo completo de autenticación (CAS, cookies, tokens JWT)
- Explorar y probar la API con OpenAPI y Scalar
- Usar `useAxios` y `HttpApi` para comunicarse con la API desde Vue
- Mostrar estados de carga (spinners, skeletons) de forma reactiva
- Conocer el patrón Suspense de Vue
